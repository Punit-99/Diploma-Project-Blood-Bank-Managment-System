step 1 : install python (add to path)
step 2 : install pip
step 3: Run "python -m pip install -r requirements.txt"
step 4: Run these commands "py manage.py makemigrations
py manage.py migrate
py manage.py runserver"
step 5 : run "py manage.py createsuperuser"
NOTE : the password is invisible just hit enter when you done 
step 6:paste this on browser "http://127.0.0.1:8000/"
step 7: Now your project is live if u wish to stop the host you can press ctrl+c on terminal
